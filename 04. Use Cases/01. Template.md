# Key Use Cases

## UC-1: Employee Submits a Leave Request
1. Employee fills out the leave form.
2. Frappe Doctype `Leave Application` is created.
3. Validation checks leave balance.
4. Workflow triggers HR Manager approval.

## UC-2: Payroll Generation
1. HR finalizes attendance.
2. Django signal triggers payroll calculation.
3. Data pushed back into Frappe for viewing.
